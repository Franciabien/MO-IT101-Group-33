## Program Details

The **MotorPH Payroll System** is a Java-based application designed to simulate a basic payroll processing system for employees. The program reads employee data from a **text file** and automatically calculates mandatory government deductions including **SSS, PhilHealth, Pag-IBIG, and Income Tax**. After computing all required deductions, the system generates a **clear and organized output** that shows the employee’s salary breakdown, total deductions, and final net pay.

The goal of the program is to demonstrate how payroll systems process employee data, apply government-mandated deductions, and present payroll results in a simple and understandable format.
